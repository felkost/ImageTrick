## Установка
  ### Linux:
  * cd Path/ImageTrick/main
  * conda create -n new_environment python=3.6
  * conda activate new_environment
  * while read requirement; do conda install --yes $requirement || pip install $requirement; done < requirements.txt
  * ./TUIprogram.sh
